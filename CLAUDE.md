# DROP — UNICEF Hackathon

Lee @DROP_context.md para entender el proyecto completo antes de hacer nada.

## Stack
Next.js (App Router) + Tailwind CSS + localStorage. Sin backend ni Auth externo.

## Estética — NO NEGOCIABLE
- Fondo: #0a0a0a
- Acentos neón: cian (#00f5ff), violeta (#a855f7), rosa (#ff0080)
- Tipografía display: Orbitron o Bebas Neue (nunca Inter, Roboto o Arial)
- Glow effects, micro-animaciones, layouts asimétricos
- Sensación de festival de música electrónica / rave

## Reglas
- Nunca diseño genérico de IA
- Siempre en español
- Componentes reutilizables desde el inicio
- Todo lo que persista va a localStorage
